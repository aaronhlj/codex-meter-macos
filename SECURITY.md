# Security Policy / 安全政策

## 报告安全问题 / Reporting a Vulnerability

请不要公开提交可能暴露凭据、会话数据或可被利用的安全问题。请通过 GitHub 仓库所有者主页提供的联系方式私下报告，并包含受影响版本、复现步骤和影响范围。收到报告后，我们会尽快确认并评估修复方案。

Please do not publicly disclose vulnerabilities that may expose credentials, conversation data, or exploitable behavior. Report them privately using the contact method listed on the repository owner's GitHub profile. Include the affected version, reproduction steps, and potential impact. We will acknowledge and assess reports as soon as practical.

## 范围 / Scope

重点关注本应用的本地文件读取、Codex app-server 通信、通知内容、登录项以及发布包完整性。Codex 或 OpenAI 服务本身的问题应提交给其官方安全渠道。

Relevant areas include local file access, Codex app-server communication, notification content, login items, and release integrity. Issues in Codex or OpenAI services should be reported through their official security channels.
