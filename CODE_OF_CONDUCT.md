# Code of Conduct / 行为准则

我们致力于建设一个友好、包容、尊重彼此的社区。参与者应以建设性的方式交流，尊重不同背景与观点，不得骚扰、侮辱、歧视或公开他人的私人信息。

We are committed to a welcoming, inclusive, and respectful community. Participants should communicate constructively, respect different backgrounds and viewpoints, and avoid harassment, insults, discrimination, or disclosure of another person's private information.

项目维护者可以删除不当内容，并对持续或严重违反本准则的参与者采取临时或永久限制。需要报告问题时，请使用仓库所有者 GitHub 主页提供的私人联系方式。

Maintainers may remove inappropriate content and temporarily or permanently restrict participants for persistent or serious violations. To report conduct concerns, use the private contact method listed on the repository owner's GitHub profile.
